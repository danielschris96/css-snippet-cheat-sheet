CSS Snippet Challenge

If you're having trouble remembering your CSS properties and attributes, refer to this document!

Flexbox Row

To create a flexbox row layout, refer to the code below:

    .row { 
        display: flex; 
        flex-direction: row;
        flex-wrap: wrap;
    }

Flexbox Column

To create a flexbox column layout, refer to the code below:

    .column { 
        display: flex; 
        flex-direction: column;
    }

CSS Grid Layout

Build a 12-column layout using CSS Grid:

    .grid { 
        display: grid; 
        width: 100%;
        grid-template-columns:
    repeat (12, 1fr);
    }

Linear Gradients

If you'd like to create a linear gradient in your css, refer to the code below:

.linear-gradient-background {
    background-image: linear-
    gradient(
                rgba(x, x, x, a) x%
                    rgba(x, x, x, a) x%
            )
        }
